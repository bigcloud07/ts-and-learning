  > 이전에 진행한 React 프로젝트에서 사용했었다. 하지만 AI의 도움을 많이 받았고 명확히 알고쓴다기 보다 구현에 딱 필요한 만큼 공부를 하고 써서 누군가 나에게 TS에 대해서 깊게 질문한다면 제대로 답변이 어려울 거 같은 정도의 수준이다. 이번 미션을 진행하면서 해당 부분에 대한 갈증을 어느정도 해소하고 싶다.

### 2. 기존에 새로운 개념을 학습할 때 어떻게 했나요? (수업시간에 v1 떠올렸던 것 첨부)
```
이번에 원정대 활동을 하며 진행한 스코프, 실행컨텍스트, 클로저의 개념 학습법입니다.

1. 모던자바스크립트 책에서 공부하고자 하는 Chapter의 내용을 쭉 읽어본다.
2. 읽으면서 모르는 부분이 있으면 머릿속에 생각 or notion에 메모만 하고 탐구하지 않고 쭉 읽어본다.
3. 다시 한 번 읽으면서 모르는 부분을 AI에게 물어보거나 크루에게 물어본다.
4. 마지막으로 읽으면서 머릿속으로 구조화한다.
5. AI에게 관련 개념 문제를 요구하고 풀어본다.
6. 틀렸다면 왜 틀렸는지 생각해보고 AI 생각과 비교해본다.
```
